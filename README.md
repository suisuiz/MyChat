# 目录
```
  common	公共的代码目录(common.js里是接口)
  components	框架封装的组件、自行封装的组件，页面里有调用
  pages	页面在此目录
  static	静态资源在此目录
  App.vue	项目入口全局监听的 websocket 和 处理其他APP传来的参数
  pages.json  所有页面都要在此定义
  manifest.json 配置name、appid

```
